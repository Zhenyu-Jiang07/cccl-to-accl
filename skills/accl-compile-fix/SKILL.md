# ACCL Compile Fix Skill

## 1. Skill 简介
本 Skill 面向华为昇腾代码迁移场景，负责将候选 ACCL 文件写入目标仓库 `mylearn`，并在指定环境中执行 host 编译验证。

如果编译失败，当前版本允许基于编译日志进行一次修复循环。

---

## 2. 第一版范围
当前版本仅支持：

- 将候选 `os.h` 写入固定目标路径
- 在 `mylearn` 中执行 host 编译
- 收集编译日志
- 最多执行一次修复循环

当前版本暂不支持：

- kernel 编译验证
- 全量项目编译验证
- 多文件联合修复
- 多轮自动修复

---

## 3. 输入
本 Skill 的输入包括：

- 候选 ACCL `os.h` 文件内容
- 目标仓库根目录
- 目标相对路径
- 编译模式
- 最大修复轮数

---

## 4. 输出
本 Skill 的输出包括：

- 编译状态
- 编译日志
- 修复说明
- 最终目标文件内容

---

## 5. 当前目标
当前版本的主要目标是：

**将候选 ACCL `os.h` 写入 `mylearn` → 执行 host 编译验证 → 输出结果**

为整个迁移原型提供最终验证能力。
